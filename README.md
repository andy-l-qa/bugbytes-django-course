# Django Tutorial Project

This repository contains my implementation of the official Django tutorial. The project was built to learn and practice the core concepts of Django web development, including models, views, templates, routing, database migrations, and the admin interface.

## 🚀 Project Overview

This project is a simple web application built by following the official Django documentation. It demonstrates the fundamentals of Django’s MTV (Model–Template–View) architecture.

The application is a basic polling system where users can:
- View available questions
- Vote on choices
- See results

The goal of this project was to understand Django’s request/response flow, ORM-based database interaction, and template rendering system.

## 🛠️ Tech Stack

- Python 3.13.3
- Django 6.0.6
- SQLite (default development database)
- HTML / CSS (Django Templates)

## 📚 Key Concepts Practiced

- Django project structure and app setup
- URL routing and request handling
- Models, migrations, and Django ORM
- Admin panel customization
- Function-based views (FBVs)
- Template rendering system
- Forms and user input handling
- Database queries using ORM
